This is an AHK script that automatically snipes cars that you choose on Forza Horizon 6
Instructions:
1. Download the latest verion of Auto Hot Key
2. Run the ahk file you downloaded
3. Open Forza and go to the auction house
4. select the car that you want to snipe and go into the auction list then go back one page.
5. Hover your curser over the seach auction button button after you've set the car you Chose
6. Press F6 to start the script
7. To stop the script press the tilde key (the squigly button under your escape key)
8. Notes:
9. Sometimes the script will glitch out if the car shows up on the auction house and is already sold, if the script starts failing and goes into pages its not supposed to close the script with "~" and restart the script
10. When you stop the script you have to reopen the file every time
11. Of course you're not going to get the cars every time it still might take some time, there hundreds of people trying to snipe cars at once
12. Tips:
13. Only snipe cars from the horizon festival not your house since theres less delay from the festival
14. Turn your graphics all the way down and turn off moving backgrounds in the accessability tab
